# Schbot для VK

Это версия исходного Telegram-бота для сообщества ВКонтакте.

## Что умеет

- `/start` — запуск и меню.
- `/record` — запись на Ментальное ГТО.
- `/del_record` — удалить запись на Ментальное ГТО по ID.
- `/admin` — список записей для администраторов.

Данные хранятся в `data.json`. Незавершённые диалоги хранятся в памяти и сбрасываются после перезапуска.

## Настройка VK

1. Создайте или откройте сообщество VK.
2. В управлении сообществом включите сообщения сообщества.
3. В разделе API создайте ключ доступа сообщества с правом на сообщения.
4. Включите Long Poll API.
5. В типах событий Long Poll включите:
   - входящие сообщения (`message_new`);
   - события callback-кнопок (`message_event`).

## Запуск

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
```

Заполните `.env`:

```env
VK_GROUP_TOKEN=...
VK_GROUP_ID=...
VK_ADMIN_IDS=...
CURRENT_YEAR=2026
```

После этого:

```bash
python bot_vk.py
```

## Важные отличия от Telegram-версии

- В `.env` нужно указывать VK ID администраторов, а не Telegram ID.
- Telegram-стикер из исходного бота не переносится в VK, поэтому вместо него отправляется текст ошибки.
- В VK нет меню команд как в Telegram, поэтому бот отправляет клавиатуру с кнопками команд при `/start`.
